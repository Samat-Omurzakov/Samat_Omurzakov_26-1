 Формы - это группы из одного или нескольких полей записанные в тег <form> ... </form>, которые представляют данные одного объекта.
Формы собирают информацию от пользователя и отправляют на сервер. В django существуют 3 вида форм: формы связанные с моделями, HTML формы, Forms классы.
    Атрибуты, которые должны присутствовать в форме:
1) action - URL, на который будут отправлены данные
2) method - HTTP метод, который должна использовать форма для отправки данных(GET или POST)
GET и POST – единственные HTTP методы, которые используются для форм.
